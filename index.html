game guide

RISE RISE!는 어떤 게임인가요?

RISE RISE!는 물이 차오르는 방 안에서 가구를 떨어뜨려 최대한 높은 탑을 만드는 2D 캐주얼 스태킹 게임입니다. 플레이어는 흔들리는 가구가 적절한 위치에 왔을 때 키를 눌러 가구를 쌓고, 제한 시간 동안 가능한 높은 기록을 만드는 것을 목표로 합니다.

처음에는 하나의 화면만 관리하면 되지만, 기록이 높아지면 화면이 2개, 3개, 4개로 분할됩니다. 화면이 늘어날수록 각 키를 번갈아 확인해야 하므로 순발력과 균형 감각이 함께 필요합니다.

기본 조작법

화면에 표시된 A, S, D, F 키를 눌러 해당 칸의 가구를 떨어뜨립니다. 가구가 아래 가구와 많이 겹칠수록 안정적으로 쌓이며, 너무 벗어나면 실패 처리되어 물이 차오르는 압박이 커집니다.

높은 기록을 만드는 방법

초반에는 넓은 가구로 안정적인 기반을 만들고, 중반 이후에는 무리한 위치보다 안전한 위치를 선택하는 것이 좋습니다. 화면이 분할되면 모든 화면을 동시에 보려 하기보다 가장 위험한 화면부터 처리하세요.

화면 분할 규칙

점수가 높아지면 화면이 여러 칸으로 나뉩니다. 각 칸은 A, S, D, F 키로 조작하며, 동시에 여러 탑을 관리해야 하므로 후반으로 갈수록 집중력이 중요합니다.

공유 기능

결과 화면에서 기록을 공유하면 내 점수가 포함된 도전 링크가 만들어집니다. 친구가 링크를 열면 같은 사이트에서 바로 도전할 수 있어 기록 경쟁을 이어가기 좋습니다.

자주 묻는 질문

모바일에서도 플레이할 수 있나요?

모바일에서는 화면 크기에 맞춰 레이아웃이 단순화됩니다. 키 입력 중심 게임이므로 데스크톱이나 키보드가 있는 환경에서 가장 안정적으로 플레이할 수 있습니다.

점수는 어떻게 계산되나요?

점수는 플레이어가 쌓아 올린 가구의 높이를 기준으로 표시됩니다. 결과 화면에서 이번 기록과 최고 기록을 확인할 수 있습니다.

광고는 어디에 표시되나요?

광고는 게임 조작 영역을 가리지 않는 좌우 세로형 영역에만 배치할 수 있도록 준비했습니다. 광고와 게임 화면 사이에는 여유 공간을 두어 실수 클릭이 발생하지 않도록 설계했습니다.
